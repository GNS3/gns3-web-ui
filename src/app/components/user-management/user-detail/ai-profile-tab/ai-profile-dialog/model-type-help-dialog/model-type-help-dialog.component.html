Basic Configuration

Configuration Name

A unique identifier for this LLM model configuration.

  • Format: 1-100 characters
  • Allowed characters: Letters, numbers, underscores, and hyphens
  • Example: gpt-4o-config, claude-3-5-sonnet

Model Type

The category of AI model you are configuring. This field is reserved for future use.

info Currently, only text model type is supported. Other options (vision, stt, tts, multimodal, embedding, reranking, other) are reserved for future enhancements.

Select LLM Provider

Choose a preset provider to auto-configure the connection settings.

  • OpenRouter: Access to multiple models through openrouter.ai
  • DeepSeek: DeepSeek's official API endpoint
  • Custom: Manually configure all settings

Presets automatically fill in the Base URL and default model list.

Model Name

The specific model to use from the selected provider.

  • From preset: Select from a dropdown list of available models
  • Custom: Enter any model name manually

Examples: gpt-4o, claude-3-5-sonnet-20241022, deepseek-chat

API Configuration

Model Provider

The provider type for API communication.

Select the provider that matches your API endpoint. For OpenAI-compatible APIs, choose "OpenAI" and customize the Base URL.

Base URL

The API endpoint URL for the provider.

  • Default: Leave empty to use the provider's default endpoint
  • Custom: Enter a full URL (e.g., https://api.openai.com/v1)

This field is auto-filled when using presets.

API Key

Your authentication key for accessing the API.

  • Create mode: Required field, minimum 10 characters
  • Edit mode: Leave empty to keep existing key

lock API keys are encrypted on the server for security.

Model Parameters

Temperature

Controls the randomness of the model's responses.

  • Range: 0.0 to 2.0
  • Lower values (0.0-0.3): More focused, deterministic output
  • Higher values (0.7-2.0): More creative, diverse output
  • Recommended: 0.3 for technical tasks, 0.7 for creative tasks

Context Limit

The maximum context window size in thousands (K) of tokens.

  • Range: 1 to 10000
  • Unit: K tokens (e.g., 128 = 128K tokens)
  • Meaning: Maximum amount of text the model can consider

Different models have different context limits. Check your model's documentation.

Context Strategy

Determines how aggressively the system uses the available context limit.

Strategy Usage
Conservative 60% of context limit (safer, more reserved)
Balanced 75% of context limit (recommended)
Aggressive 85% of context limit (maximizes capacity)

Copilot Mode

Determines the AI assistant's operational permissions within GNS3.

Mode Capabilities
Teaching Assistant Diagnostics and suggestions only (read-only)
Lab Automation Assistant Full configuration access (can modify settings)

Advanced Parameters (Optional)

Add custom parameters to pass additional options to the API.

  • Key: Parameter name (e.g., max_tokens)
  • Value: Parameter value (e.g., 4096)

This field is reserved for future use. Additional parameters may be supported in later versions.

Default Configuration

When enabled, this configuration will be used as the default for AI Chat and other AI-powered features.

Only one configuration can be set as default at a time.
