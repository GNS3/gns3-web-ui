help_outline Model Type Documentation

Overview

The Model Type field specifies the category of AI model you are configuring. This field is reserved for future use and helps the system understand how to handle different model capabilities.

Current Status

info Currently, only the text model type is fully supported by the backend. Other options are reserved for future enhancements.

Available Model Types

Type Description Status
text Text generation models (e.g., GPT-4, Claude) ✓ Supported
vision Image understanding models Reserved
stt Speech-to-Text models Reserved
tts Text-to-Speech models Reserved
multimodal Multiple input types (text, image, audio) Reserved
embedding Text embedding models Reserved
reranking Reranking models Reserved
other Other specialized model types Reserved

Recommendation

For now, always select text as the model type. The system will automatically handle other model types when they become available in future updates.
