## 升级策略

### 方案一：逐个升级
1. 先升级 Angular 核心包
2. 逐个升级第三方库
3. 处理冲突

### 方案二：使用 --legacy-peer-deps
```bash
npm install @angular/core@16 @angular/material@16 @angular/cdk@16 --legacy-peer-deps
```

### 方案三：寻找替代库
如果某些库不支持 Angular 16：
- 寻找社区 fork 版本
- 自行修改兼容
- 使用原生方案替代
