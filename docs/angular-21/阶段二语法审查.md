## 修复指南

### 1. 移除 standalone: true

**问题**: Angular 20+ 默认为 standalone，不需要显式声明

**修复前**:
```typescript
@Component({
  selector: 'app-example',
  standalone: true,  // 删除这行
  ...
})
```

**修复后**:
```typescript
@Component({
  selector: 'app-example',
  ...
})
```

### 2. styleUrls → styleUrl

**问题**: 当只有一个样式文件时，使用单数形式

**修复前**:
```typescript
@Component({
  styleUrls: ['./example.component.scss'],
  ...
})
```

**修复后**:
```typescript
@Component({
  styleUrl: './example.component.scss',
  ...
})
```

### 3. @HostListener → host

**问题**: 装饰器方式已过时

**修复前**:
```typescript
@HostListener('window:resize', ['$event'])
onResize(event: Event) { ... }
```

**修复后**:
```typescript
@Component({
  host: {
    '(window:resize)': 'onResize($event)'
  },
  ...
})
```

### 4. @HostBinding → host

**修复前**:
```typescript
@HostBinding('class.active')
isActive = false;
```

**修复后**:
```typescript
@Component({
  host: {
    '[class.active]': 'isActive'
  },
  ...
})
```
