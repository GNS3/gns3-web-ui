## 迁移建议

### 优先级顺序

1. **第一优先级：简单对话框**
   - 剩余少量对话框组件
   - 复杂度低，易于迁移

2. **第二优先级：通用组件**
   - Preferences 中的通用组件
   - 节点配置器组件

3. **第三优先级：中等复杂度组件**
   - AI Chat 组件
   - 用户/组/角色管理组件

4. **第四优先级：大型页面组件**
   - ProjectMapComponent
   - ControllersComponent
   - Cartography 组件

### 迁移步骤

1. 分析组件依赖
2. 添加必要的 Material 模块到 imports
3. 使用 `inject()` 替代构造函数注入
4. 从 declarations 移到 imports
5. 测试构建
