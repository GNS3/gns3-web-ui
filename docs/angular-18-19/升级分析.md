**当前**: TypeScript 5.5.x
**需要**: TypeScript 5.6+

```json

// 修改后
```

## 5. 主题系统

**文件**: `src/styles/theme-v2.scss`

### Angular 19 可能的变化：

1. **M2 API 变化**
   - 继续使用 `m2-define-palette`
   - 可能需要调整

2. **主题配置**
   - 可能需要更新 `define-theme` API

### 预期修复：

```scss
// Angular 18 (当前)
$gns3-primary: mat.m2-define-palette(mat.$m2-cyan-palette, 700, 500, 900);

// Angular 19 (可能需要)
$gns3-primary: mat.m2-define-palette(mat.$m2-cyan-palette, 700, 500, 900);
```

## 7. 风险评估

### 低风险

## 9. Angular 19 新特性（可选）

### Signal API

Angular 19 推荐使用 Signal 替代部分 RxJS：

```typescript
// 当前方式 (仍然支持)
name = new FormControl('');

// Signal 方式 (推荐)
name = signal('');
```

### 控制流语法

```html
<!-- 当前方式 (仍然支持) -->
<div *ngIf="show">Content</div>

<!-- 新方式 (推荐) -->
@if (show) {
  <div>Content</div>
}
```

## 10. 总结

### 升级可行性：✅ 推荐

1. **主要变更**: TypeScript 5.6+ 要求
2. **无需大规模重构**: Forms 和 NgModule 仍然兼容
3. **可选迁移**: Signal API 和新控制流语法
4. **风险评估**: 低风险升级
